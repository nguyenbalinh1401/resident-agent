# Pulse - Intelligent Resident Services & Management System

## Overview

Pulse is an intelligent building management system with AI conversational interface, replacing traditional forms with natural language interactions.

## Features

### 🏢 Building Management (Admin/Staff)

#### Building & Unit Management
- View building overview (Block → Floor → Unit)
- Manage unit status
- Assign residents to units

#### Financial Management
- Create and manage bills
- Track resident debts
- Configure fee types and payments

#### Operations Management
- Receive and process support requests
- Schedule maintenance tasks
- Manage IoT devices in building

#### Amenity Management
- Configure amenities (pool, gym, etc.)
- Set operating hours
- Approve/Reject booking requests

#### Package Management
- Log incoming packages
- Update delivery status

#### Communication
- Send announcements
- Create surveys
- View system activity logs

### 👤 Resident

#### Personal Information
- View profile and unit details
- Update contact information
- Register personal vehicles

#### Payments
- View bills and outstanding balances
- Pay online (VNPay/MoMo)
- View payment history

#### Amenity Booking
- View available amenities
- Book time slots (pool, gym, meeting room...)
- Cancel bookings

#### Support Requests
- Report incidents/maintenance issues
- Track request status
- Chat with building management

#### Visitor Management
- Register visitors
- Receive arrival notifications

#### Notifications
- Building announcements
- Payment reminders
- Package arrival alerts

## AI Integration

- **Conversational Interface**: Natural language interactions
- **CUX Orchestrator**: Intent recognition & workflow automation
- **Auto-Execution**: Automatically create requests from conversations
